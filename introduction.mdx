## What is nexDash?

nexDash is a full-featured SaaS dashboard application built with Next.js 14, designed to help you manage customers, track invoices, and visualize revenue data. Built with modern web technologies and best practices, nexDash provides a solid foundation for building production-ready business applications.

## Key features

<AccordionGroup>
  <Accordion title="Customer management" icon="users">
    Comprehensive customer database with profile management, contact information, and invoice history tracking. Built with server-side rendering for optimal performance.
  </Accordion>

  <Accordion title="Invoice tracking" icon="file-invoice">
    Create, edit, and manage invoices with status tracking (pending/paid). Includes automatic calculations and customer associations.
  </Accordion>

  <Accordion title="Revenue analytics" icon="chart-line">
    Visual revenue charts and analytics dashboard powered by real-time data from Vercel Postgres. Track monthly trends and financial metrics.
  </Accordion>

  <Accordion title="Secure authentication" icon="shield-check">
    Built-in user authentication with bcrypt password hashing. Includes protected routes and session management.
  </Accordion>

  <Accordion title="Modern tech stack" icon="layer-group">
    Next.js 14 with App Router, TypeScript, React 18, Tailwind CSS, and Vercel Postgres for a production-ready foundation.
  </Accordion>
</AccordionGroup>

## Core capabilities

### Dashboard overview

The main dashboard provides a comprehensive view of your business metrics:

- Total revenue with trend indicators
- Invoice status summary (pending vs. paid)
- Customer count and recent activity
- Monthly revenue chart visualization

### Data management

nexDash includes robust data management features:

<Steps>
  <Step title="Customers">
    Manage customer profiles with name, email, and profile images. Each customer has a detailed view showing all associated invoices.
  </Step>
  <Step title="Invoices">
    Track invoices with amount, status, date, and customer association. Filter and search capabilities included.
  </Step>
  <Step title="Revenue">
    Monthly revenue tracking with visual charts. Data is stored and retrieved efficiently from Postgres.
  </Step>
</Steps>

## Sample data included

nexDash comes with placeholder data to help you get started quickly:

- 10 sample customers with profile images
- 15 sample invoices with various statuses
- 12 months of revenue data
- Demo user account for testing authentication

You can seed this data to your database with a single command (covered in the quick start guide).
