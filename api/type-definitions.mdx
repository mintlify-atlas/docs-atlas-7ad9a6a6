This module contains TypeScript type definitions for all data models in the application. These types ensure type safety when working with database records, API responses, and form data.

## User

Represents a user account in the system.

```typescript
export type User = {
  id: string;
  name: string;
  email: string;
  password: string;
};
```

<ResponseField name="id" type="string">
  Unique identifier for the user.
</ResponseField>

<ResponseField name="name" type="string">
  Display name of the user.
</ResponseField>

<ResponseField name="email" type="string">
  Email address of the user (used for authentication).
</ResponseField>

<ResponseField name="password" type="string">
  Hashed password for the user account.
</ResponseField>

## Invoice

Represents an invoice record with core fields.

```typescript
export type Invoice = {
  id: string;
  customer_id: string;
  amount: number;
  date: string;
  status: 'pending' | 'paid';
};
```

<ResponseField name="id" type="string">
  Unique identifier for the invoice.
</ResponseField>

<ResponseField name="customer_id" type="string">
  Foreign key reference to the customer who owns this invoice.
</ResponseField>

<ResponseField name="amount" type="number">
  Invoice amount in cents.
</ResponseField>

<ResponseField name="date" type="string">
  ISO date string representing when the invoice was created.
</ResponseField>

<ResponseField name="status" type="'pending' | 'paid'">
  Current payment status of the invoice. This is a string union type that can only be 'pending' or 'paid'.
</ResponseField>

## LatestInvoiceRaw

Represents a recent invoice as returned from the database before formatting.

```typescript
export type LatestInvoiceRaw = Omit<LatestInvoice, 'amount'> & {
  amount: number;
};
```

This type extends `LatestInvoice` but with the amount field as a number instead of a formatted string. The database returns a number for amount, which is later formatted to a string using the `formatCurrency()` function.

<ResponseField name="amount" type="number">
  Numeric invoice amount in cents (before formatting).
</ResponseField>

## FormattedCustomersTable
