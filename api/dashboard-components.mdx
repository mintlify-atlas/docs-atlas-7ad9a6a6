## Card

Displays a metric card with an icon, title, and value. Used to show key statistics like collected revenue, pending invoices, total invoices, and customer count.

<ParamField path="title" type="string" required>
  The title of the card displayed at the top
</ParamField>

<ParamField path="value" type="number | string" required>
  The metric value to display (can be a number or formatted string)
</ParamField>

<ParamField path="type" type="'invoices' | 'customers' | 'pending' | 'collected'" required>
  The card type which determines the icon displayed
</ParamField>

### Usage

```tsx
import { Card } from '@/app/ui/dashboard/cards';

export default function Dashboard() {
  return (
    <Card 
      title="Collected" 
      value={totalPaidInvoices} 
      type="collected" 
    />
  );
}
```

## CardWrapper

Async wrapper component that fetches and displays multiple metric cards in a grid layout.

### Usage

```tsx
import CardWrapper from '@/app/ui/dashboard/cards';

export default function Dashboard() {
  return (
    <div className="grid gap-6 sm:grid-cols-2 lg:grid-cols-4">
      <CardWrapper />
    </div>
  );
}
```

## RevenueChart

Displays a bar chart visualization of monthly revenue data. Shows the last 12 months of revenue with responsive layout.

<ParamField path="revenue" type="Revenue[]" required>
  Array of revenue data objects containing month and revenue amount
</ParamField>

### Revenue type

```typescript
type Revenue = {
  month: string;
  revenue: number;
};
```

### Usage

```tsx
import RevenueChart from '@/app/ui/dashboard/revenue-chart';
import { fetchRevenue } from '@/app/lib/data';

export default async function Dashboard() {
  const revenue = await fetchRevenue();
  
  return (
    <RevenueChart revenue={revenue} />
  );
}
```

## LatestInvoices

Displays a list of the most recent invoices with customer information, including profile pictures, names, emails, and amounts.

<ParamField path="latestInvoices" type="LatestInvoice[]" required>
  Array of invoice objects to display
</ParamField>

### LatestInvoice type

```typescript
type LatestInvoice = {
  id: string;
  name: string;
  image_url: string;
  email: string;
  amount: string;
};
```

### Usage

```tsx
import LatestInvoices from '@/app/ui/dashboard/latest-invoices';
import { fetchLatestInvoices } from '@/app/lib/data';

export default async function Dashboard() {
  const latestInvoices = await fetchLatestInvoices();
  
  return (
    <LatestInvoices latestInvoices={latestInvoices} />
  );
}
```

## SideNav

Main navigation sidebar component that includes the logo, navigation links, and sign-out button. Responsive design that adapts for mobile and desktop.

### Usage

```tsx
import SideNav from '@/app/ui/dashboard/sidenav';

export default function Layout({ children }) {
  return (
    <div className="flex h-screen flex-col md:flex-row md:overflow-hidden">
      <div className="w-full flex-none md:w-64">
        <SideNav />
      </div>
      <div className="flex-grow p-6 md:overflow-y-auto md:p-12">
        {children}
      </div>
    </div>
  );
}
```

## NavLinks

Client-side navigation links component that highlights the active page based on the current pathname. Displays links for Home, Invoices, and Customers pages.

### Usage

```tsx
import NavLinks from '@/app/ui/dashboard/nav-links';

export default function SideNav() {
  return (
    <div className="flex grow flex-row justify-between space-x-2 md:flex-col md:space-x-0 md:space-y-2">
      <NavLinks />
    </div>
  );
}
```
